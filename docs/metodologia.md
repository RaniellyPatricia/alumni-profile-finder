# Metodologia inicial

Este projeto tem como objetivo desenvolver um protótipo em Python para busca assistida e organização de perfis profissionais públicos.

## Escopo do MVP

A primeira versão do projeto será testada apenas com perfil próprio ou com perfis autorizados.

O sistema deverá:

1. Receber dados de entrada, como nome, instituição, curso e ano de entrada.
2. Montar consultas de busca para localizar possíveis perfis públicos.
3. Organizar os possíveis resultados encontrados.
4. Gerar um arquivo CSV para análise e validação manual.
5. Evitar coleta de dados privados ou sensíveis.

## Cuidados éticos

O projeto não tem como objetivo burlar login, captcha, bloqueios ou restrições de plataformas.

A validação dos perfis será manual ou semiautomática.

Os dados reais não devem ser publicados no GitHub.
